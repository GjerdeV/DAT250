For å kjøre Flask applikasjonen/serveren må en først angi hvilken python fil som er hoved fila.
Dette gjøres ved å kjøre følgende kommando (Flask serveren ser etter app.py som default/standard):

windows powershell: $env:FLASK_APP = "app.py"
windows cmd:        set FLASK_APP=app.py
unknown:            export FLASK_APP=app.py

Deretter kan følgende kommando kjøres (men bare etter at venv er aktivert i vs code), 
denne kommandoen vil prøve å kjøre den fila som vi viste til med kommandoene ovenfor:

windows: python -m flask run
mac: python3 -m flask run

server alternativer:

disse er trygge:
  python -m flask run
  flask run

denne gjør slik at andre enheter på nettverket (med litt ekstra konfigurasjon, også internet) kan få tilgang til serveren,
problemet me dette er at hvis feil person vet om det så kan de kjøre vilkårlig python (skadelig) kode på maskinen din:
flask run --host 0.0.0.0
